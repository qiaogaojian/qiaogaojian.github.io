    Windows 配置环境变量

    Settings > System > AdvanceSystemSetting > Environment Variables > Path > Edit

    git config --global user.name "qiaogaojian"  
git config --global user.email "qiaogaojian@vip.qq.com"

    ssh-keygen -t rsa -C "qiaogaojian@vip.qq.com"
