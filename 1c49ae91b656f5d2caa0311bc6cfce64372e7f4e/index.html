    n 常用命令有哪些?

    # 安装 11.5.0 版本的 Node.js 
n 11.5.0                   

# 安装稳定版 nodejs
sudo n stable 

# 安装最新版 nodejs
sudo n latest

# 删除某个版本
n rm 10.13.0

# 使用 n 切换版本
n   --回车
    node/10.13.0
  ο node/10.15.3
    node/11.0.0
    node/11.8.0
    node/12.2.0
# 按上下键选择版本后，回车

# nodejs 版本
node -v

# 以指定的版本来执行脚本
n use 11.5.0  test.js
